Vehicle Control Unit
====================
The Vehicle Control Unit is the processor on the car responsible for taking driver inputs, communicating with the inverter, and 
the traction control unit.

Resources
#########
* `Modern Embedded Systems <https://youtube.com/playlist?list=PLPW8O6W-1chwyTzI3BHwBLbGQoPFxPAPM>`_: YouTube course on embedded programming (Arm).
* `RTOS Fundamentals <https://freertos.org/implementation/a00002.html>`_: A short recap of RTOS concepts. Not suitable for beginners.
